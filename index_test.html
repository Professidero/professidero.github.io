• StreamLauncher
    • Description
    • Gallery

    Open a stream on your favorite desktop video player (like VLC) and manage all your favorite streams easily


    • GUI for livestreamer
    • Open a stream without dealing with any system console
    • Manage a list of your favorite streams
    • Open a stream chat in your browser from this software

    Context : Personal project

    Type : GUI for a CLI-based software

    Platform : PC, tested on Windows 7

    When was it made : 2015

    Team : Alone

    Technology : Ruby and Shoes

    Source : Github repository


    Note : Later, another version was made but is not even quite achieved. It is available in this repository


    Note 2 : From now on, Twitch streams can't be opened anymore by livestreamer without a token and livestreamer is no longer maintained, resulting in my program not being able to do his main job anymore

  • Demonio
    • Description
    • Gallery

    Choose your fighter to explore a maze and kill all the monsters !


    • You earn gold and experience when you killed an enemy
    • You can move in the maze using number as an input
    • When an enemy is near you, he will start chasing you
    • Fights are automatically solved

    Context : University small project

    Type : Rogue-like (turn-based)

    Platform : System console, tested on Windows console

    When was it made : 2013

    Team : Alone

    Technology : Java (with Eclipse IDE)

    Source : Github repository
