    Video game version of the French board-game named "Jeu des Petits Chevaux"


    • 2 to 4 players, with a 6-side dice, 4 horses (all in their stable at the beginning of the game) per player
    • The first player to reach the last square wins
    • The number you get on your die roll is the number of squares a horse of your choice can cross
    • Get a 6 to get an horse out of the stable

    Context : University small project

    Type : Board-game (turn-based)

    Platform : PC, tested on Windows

    When was it made : 2016

    Team : 2

    What did I do : The tasks were not clearly shared so I did various things from representation of the checkerboard' squares to horse movements, and a small bit of the GUI

    Technology : Java (with Eclipse IDE), the GUI is made with Java using AWT and Swing

    Source : Github repository


    Note : The GUI wasn't made in time so there is a board but no horse moving on it, the movement's feedbacks are made with some text
