    Discovering libGDX framework through the making of a Pac-Man game


    • Pac-Man can navigates through the maze using keyboard and eat Tic-Tacs
    • Tic-Tacs increase the player score when collected
    • There is 3 ghosts who can kill Pac-Man, each one having a different navigating behaviour
    • Collect all the Tic-Tacs to win the game

    Context : University small project. Everything was coded during 4 practical work sessions (each session was 4 hour long)

    Type : Maze

    Platform : PC, tested on Windows

    When was it made : 2017

    Team : Alone

    Technology : Java and libGDX framework (with Eclipse IDE and Android Development Tools plugin)

    Source : Github repository
