    Solve the travelling salesman problem using simulated annealing algorithm


    • Build a simplified roadmap
    • Try to find a solution to the TSP using the annealing algorithm
    • Print the graph and the solution

    Context : University small project. Most of the classes were programmed during previous practical works and the graph visualization tool (bsplines) was provided by our teacher so my work was all about designing and programming what ended up being the CircuitGraphe class (and the Make file)

    Platform : Linux, tested on Debian Jessie

    When was it made : 2017

    Technology : C++ (with gcc and Make)
